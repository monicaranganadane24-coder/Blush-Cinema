# Mini-Projet : Cinéma

Application web permettant aux clients d’un cinéma de village d’explorer les films disponibles à l’affiche.  
Le projet utilise l’API **OMDb (The Open Movie Database)** pour récupérer les informations des films.


## Objectifs
- Attirer un public plus jeune grâce à une interface moderne et accessible.
- Permettre la **visualisation des films tendances**, la **recherche en temps réel**, et l’**affichage des détails complets** d’un film.
- Respecter les bonnes pratiques de développement web : responsive design, accessibilité, modularisation du code, gestion Git.



## Structure du projet
Le site contient **3 pages principales** :

- index.html : Page d’accueil affichant les films tendances (poster, titre, résumé, bouton *Learn more*).  
- search.html : Page de recherche avec résultats instantanés (sans rechargement de page).  
- movie.html : Page de détails d’un film (titre, poster, résumé, genre, acteurs, notes, date de sortie DVD).


## Outils utilisées
- **HTML / CSS** : structure et design (thème sombre, néon bleu, responsive).  
- **JavaScript** : logique de recherche, affichage dynamique, gestion des événements.  
- **OMDb API** : récupération des données des films.  
- **Bundler** : organisation des fichiers JS par page.  
- **Git & GitHub** : gestion du code source, branches et commits.
